# Praxis-Theorie-Verknüpfung im Sportunterricht

Moderne Next.js/Tailwind-Einseiten-Webpage für das Leitprinzip **Praxis-Theorie-Verknüpfung im Sportunterricht**.

## Entwicklung

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Inhalte

- Helles Premium-Theme mit Glassmorphism
- Sticky Header mit Link zu [Sportunterricht.eu](https://sportunterricht.eu)
- Klickbare, zoombare und druckbare Infografik-Lightbox
- Kompetenzorientierte Inhaltskarten
